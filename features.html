توثيق ميزات نظام ERP Market المتكامل

تحليل مفصل ودقيق لكافة ميزات النظام، من نقاط البيع والمبيعات إلى المشتريات والنظام المحاسبي المتطور.

نظام المبيعات ونقاط البيع (Sales & POS)

نقطة البيع (POS System)

  • دعم الباركود الذكي (Scanner) ونظام منع التكرار.
  • البحث اليدوي عبر الاسم، الكود، أو الباركود.
  • سلة مبيعات فورية مع حساب تلقائي للضريبة.
  • إدارة العملاء (نقدي أو ذمم).
  • دفع متعدد (نقد، بطاقة، تحويل).
  • طباعة حرارية وتلقائية للفواتير.

إدارة الفواتير والمبيعات

  • سجل كامل للمبيعات وحالات الدفع.
  • تفاصيل دقيقة لكل فاتورة سابقة.
  • فوترة رسمية للشركات (B2B).

نظام المشتريات (Purchases Module)

أوامر الشراء

  • واجهة إدخال ديناميكية للموردين والأصناف.
  • تحديد التكلفة والكمية لكل صنف.
  • زيادة المخزون تلقائياً عند الترحيل.
  • إنشاء قيود محاسبية آلية (مخزون -> موردين).

إدارة الموردين

  • قاعدة بيانات شاملة لبيانات الاتصال.
  • متابعة كشف حساب المورد والأرصدة.

النظام المحاسبي والمالي (Accounting)

شجرة الحسابات والقيود

  • هيكل شجري مرن (أصول، خصوم، إيرادات، مصروفات).
  • قيود آلية للعمليات (بيع، شراء).
  • قيود يدوية للتسويات المعقدة.

السندات والتقارير

  • سندات قبض وصرف مبسطة مع ترحيل محاسبي دقيق.
  • ميزان مراجعة للتحقق من التوازن.
  • قائمة الدخل (الأرباح والخسائر).
  • الميزانية العمومية والمركز المالي.

ميزات تقنية وعامة

تعدد اللغات (i18n)

  • دعم كامل للعربية (RTL) والإنجليزية (LTR).
  • تغيير فوري للغة دون إعادة تحميل.

الأمان والصلاحيات

  • نظام أدوار (Admin, Manager, Accountant, Cashier).
  • صلاحيات محددة لكل دور وظيفي.

الأداء والسرعة

  • تقنية Lazy Loading لتحميل سريع.
  • نظام خفيف وسريع الاستجابة (SPA).

المتجر الإلكتروني

  • واجهة طلب للعملاء أونلاين.
  • تزامن فوري للمخزون.
  • لوحة تحكم خاصة للطلبات.

ملخص سير العمليات

1. دورة المبيعات: طلب -> مسح باركود -> دفع -> فاتورة -> خصم مخزون -> إيراد.

2. دورة المشتريات: طلب مواد -> فاتورة شراء -> زيادة مخزون -> استحقاق مورد.

3. دورة المحاسبة: تجميع عمليات -> ترحيل قيود -> قوائم مالية.
